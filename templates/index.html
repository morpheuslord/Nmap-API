This Web-page is a direct implementation of our Nmap-API it functions by taking user input the input is forwarded to the API and a json output is received which is converted to html.

The Aim is to create an effective API that can be used for higher end applications by multiple uses as the currently available API's are low powered compared to what an actual application can achieve.

We are implementing a easy to use api which has multiple profiles or scan options which uses nmap program to conduct scans and export the scans over the internet via API for more innovative and effective use, the API is easy to implement and can run on all Operating systems or dev environments as the API itself is running on a linux server doing most of the intense work.

The implementation part and the API working is explained in the documentation section of this web page.
